SERVER RULES

RULE 01
荒らし行為の禁止

サーバーの運営を妨げるあらゆる荒らし行為を禁止します。 スパム送信、権限の悪用、過度なメンション、またはサーバーの安定性を損なう行為が確認された場合、即座に処罰の対象となります。

RULE 02
不快な行為の禁止

他者が嫌がる言動や、誹謗中傷、差別的な発言を禁止します。 開発を楽しむすべてのメンバーが、心地よく過ごせる環境づくりにご協力ください。

RULE 03
運営の指示への遵守

サーバー内では、運営チーム（Owner / Admin）の指示に従ってください。 ルールに明文化されていない事象についても、運営の判断により対応を行う場合があります。
